This page shows how XML-RPC can be used to solve some user-interface problems with repetitive commenting (Often referred to as 'discussion').

I hope you post at least once, to check if it works, and to let me know how things can be improved. When everything seems to work the way it should, I'll add this to all the pages on this site.

I also want to know if this works better than commenting on Slashdot for example, and if not, what should be changed so it will be better.

You must use Internet Explorer 5 or higher, or a Mozilla based browser.
