Layout test page

The problem for me

Expecting it to be a one hour job, I tried to redo my site template without using tables. But I couldn't even get close. Getting the heights right of the sidebar and the main area was impossible. And the mixture of pixel borders and em sizes doesn't make it easier. I've used CSS for more than 2 years now, but not so much for layout, so maybe I don't know all the tricks yet.

The challenge for you

So I challenge you to give it a try too. My guess is that it's probably not possible to create an exact copy of the page. Forget about the tables for the tabs and sidebar buttons. My problem is mainly the overal layout of the page. So the challenge is: to recreate this page without using html tables with the least caveats.

A special page?

To keep things simple I created this page which has the CSS included and all scripts removed. This page is completely self-contained.
