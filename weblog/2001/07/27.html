CSS3 module: The Box Model Working Draft Published. 26 July 2001: The CSS Working Group has released the first Working Draft of CSS3 module: the box model. Part of the Cascading Style Sheets (CSS) language Level 3, this module describes the layout of textual documents in visual media. Comments are invited. Visit the CSS home page. (News archive) [The World Wide Web Consortium]

There are some interesting new features, like the :expanded and :collapsed pseudo-classes, which can be used to create outliners, without any scripting.
