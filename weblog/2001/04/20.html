 Friday, April 20, 2001

The OPML loader showed that the IE XML parser is not perfect. Compatibility issues, security issues, not to mention the 'IE only' issue. So I created a different solution. XML is parsed on the server. The data is then loaded into the page through an IFRAME with javascript data. To test this I created a Google search page. It requests the search results in XML format from Google, and then renders it.
