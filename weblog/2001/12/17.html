Joepie!
Dave nominated my weblog for the Scripting News Awards in the category Scripting Weblog! Of course I updated the button script to help you remember to vote for me!

Now I wish I had updated more frequently lately. Currently I've only got scripting and not much weblog. This really makes me update more often, if there are so few scripting weblogs. Scripting should be more popular and more widely discussed! If you script anything, no matter how simple, put it on your weblog. Even the simple scripts can teach scripting style and nice ideas.

Maybe a lot of webloggers consider posting code on their weblog not done. You don't do it in normal conversation either. For me it's kind of natural, because I write more code than I speak and write Dutch and English together. In papers I scroll to the code first, before reading the text. Let's talk code!
