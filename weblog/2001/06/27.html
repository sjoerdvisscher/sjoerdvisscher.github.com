 Wednesday, June 27, 2001

On O'Reilly, an interview with David Stutz of Microsoft, outlining terms under which they will release source code for central parts of their Dot-Net technology, on terms that allow cloning based on access to the source code. It's a stunning development, quite bold, and I'm sure we'll be thinking and talking about this for quite some time to come. [DaveNet]

Microsoft is going to release the source of "the ECMAscript compiler, written in C#, which runs on both platforms (Windows, FreeBSD)". It'll be interesting to see if that code is understandable. Could be fun to create my own language in some spare time. :-)
