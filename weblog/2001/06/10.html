Dave Winer: What is Dot-Net? I think I can say it more clearly now than before. It's a scripting environment that supports SOAP. How does it compare to others? What are the editors like? A list of features of the runtime? How does it connect to databases? Can applications access each others' storage? Simon Fell says I'm full of it. "Dotnet is strongly typed and compiled, not like any scripting environment I've worked with."

Dave is not full of it. There's JScript.net, which is ECMAScript with classes. JScript.net definitely is a scripting language, although you can compile it if you want, and let it issue al kinds of warnings you'd expect form a strongly typed language.


The smart tag code breaks Radio UserLand's display in K-Meleon. I'm sorry, Mark, the next time I'll put this stuff on a separate page. K-Meleon doesn't like the html tag in the middle of a page, although I guess a proper browser should just ignore it. It works in IE and the latest build of Mozilla. The stupid reason I needed an html tag in the middle of a document? IE only recognizes namespace definitions on an html tag, but apparently it doesn't matter if it's just an empty extra html tag.
