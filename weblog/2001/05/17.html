This is some random article from the International Herald Tribune. It's the first time I've seen a good dynamic multi-column layout.

Edge Side Includes is a "simple markup language used to define Web page components for dynamic assembly and delivery of Web applications at the edge of the Internet." [Scripting News]

There are some big names on the endorser list, like IBM, Oracle and Macromedia. Of course I agree with them that dynamic assembly at the edge of the Internet (ie. a browser) is a good idea. That's what this site is all about. But tekst-based inclusion seems a bit old-fashioned and undynamic. Using XML-RPC at the client is a much better solution IMHO.
